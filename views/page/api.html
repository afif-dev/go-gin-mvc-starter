Articles API build with MVC architectural pattern

  1. GET: articles

    curl http://localhost:8080/api/articles?page=1

  2. GET: article by Id

    curl http://localhost:8080/api/article/1

  3. POST: add article. form-data (title, short_desc, content, tags)

    curl -X POST --form "title=Test Title" --form "short_desc=Test short desc" --form "content=Test content" --form "tags=test,api,mvc,go,gin" http://localhost:8080/api/article

  4. PUT: update article. form-data (id, title, short_desc, content, tags)

    curl -X PUT --form "id=1" --form "title=Update Title" --form "tags=test,api,mvc,go,gin" http://localhost:8080/api/article

  5. DELETE: delete article. form-data (id)

    curl -X DELETE --form "id=4" http://localhost:8080/api/article
